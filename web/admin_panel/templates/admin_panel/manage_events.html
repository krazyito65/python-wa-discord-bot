Temperature Conversion Event

This event automatically detects temperature values mentioned in chat and provides conversions between Fahrenheit and Celsius.

Examples:
  • It's 75°F outside → Bot replies with Celsius conversion
  • The CPU is running at 65C → Bot replies with Fahrenheit conversion
  • Water freezes at 32 degrees F → Bot replies with Celsius conversion
Supported Formats:
  • 75F, 75°F, 75 F, 75 degrees F
  • 23C, 23°C, 23 C, 23 degrees C
